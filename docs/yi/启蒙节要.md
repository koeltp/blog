# 以钱代蓄法
以钱三文熏于炉上，致敬而祝曰：天何言哉，叩之即应；神之灵矣，感而遂通。今有某姓有事关心，不知休咎，罔释厥疑，唯神唯灵，若可若否，望垂昭报。祝毕掷钱，一背为单画“⚊”​，二背为拆画“⚋”​，三背为重画“O”​，三字为交画“×”​，自下装上。三掷内卦成，再祝曰：某宫三象，吉凶未判，再求外象三爻，以成一卦，以决犹疑。祝毕复如前法，再掷，合成一卦而断吉凶，至敬至诚无不感应。诀曰：两背由来拆，双眉本是单，浑眉交定位，总背是重安；单单单曰乾，拆拆拆曰坤，单拆单曰离，拆单拆曰坎，余卦仿此。三背为重，三字为交，重交之爻谓之发动。重作单属阳，交作拆属阴。凡动爻有变，重变拆，交变单。余爻仿此。

# 安世应和寻宫
##### 世诀
天同二世天变五，地同四世地变初。
本宫六世三世异，人同游魂人变归。
##### 应诀
往上数隔着两爻
##### 寻宫诀
一二三六外卦宫，四五游魂内变更，归魂内封是宫。
![安世应诀](img/yi/世应诀.png "世应诀")

六十四卦是由两个经卦组合而成，形成内卦与外卦的结构。在每一卦中，初爻代表内卦的地，二爻代表内卦的人，三爻则代表内卦的天。同样，四爻对应外卦的地，五爻对应外卦的人，而六爻则对应外卦的天。

“天同二世”的规则指出，当内卦与外卦的天爻阴阳属性相同时，而人爻和地爻的阴阳属性不同时，该卦的世爻位于二爻。以乾宫的《天山遁》为例，天爻一样，人爻与地多爻都不一同。因此《天山遁》的世爻在二爻，应爻则在隔两位的四爻。其它依此类推。

# 六兽歌
甲乙起青龙，丙丁起朱雀，戊日起勾陈，己日起螣蛇，庚辛起白虎，壬癸起玄武。
以日干（占卜当天的天干）决定第一爻（初爻）的六兽，如：甲日乙日起卦，第一爻为青龙。丙日丁日起卦，第一爻为朱雀。依此类推。

从初爻到上爻，按青龙 → 朱雀 → 勾陈 → 螣蛇 → 白虎 → 玄武的顺序依次往上排。

# 安月卦身诀
>阴世则从午月起，阳世还从子月生。
欲得识其卦中意，从初数至世方真。

##### 用法
看世爻的阴阳（以爻位阴阳为准，非纳甲干支阴阳）
世爻为阳爻（—）：从子月（农历十一月）开始顺数
世爻为阴爻（- -）：从午月（农历五月）开始顺数
从初爻往上数到世爻，数到第几爻，就对应第几个月
例如：世在二爻，则数2个月；世在三爻，则数3个月；世在五爻，则数5个月
从起始月（子或午）开始顺推，推出对应的地支，该地支所在的爻位即为卦身
##### 举例
火风鼎卦：
世在二爻，世爻为阳爻（▅▅▅▅▅ 辛亥官鬼）→ 阳世
阳世从子月起，数到二爻（2个月）：子 → 丑
卦身为丑，找到卦中纳甲地支为丑的爻位
初爻：辛丑 子孙 → 卦身在初爻
##### 卦身的作用
>卦身之爻为所占事之主，若无卦身则事无头绪，倘卦身有伤，其事难成矣。
+ 有卦身：事情有着落、有主体、有头绪
+ 无卦身：卦中找不到卦身地支 → 事无头绪，或此事尚未成形
+ 卦身受伤：卦身所临之爻被月建、日辰、动爻冲克 → 事情难成
