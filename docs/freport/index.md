# 财报阅读入门
## 关于财报的两个基础知识
### 时间
- 一季报：在一季度结束后一个月内披露一季度报告，但一季度报告不得早于年报。
- 半年报：在8月底之前披露半年报告。
- 三季报：在10月底披露三季报。
- 年报：公司必须在年度结束后4个月内发布财报，即最迟必须在每年的4月30日前发布该公司年报。
年报必须经会计师事务所审计并出具审计意见。所以年报准备时间最充分，信息披露最详细，可信度最高；半年报次之；一季报和三季报相对简单，仅供参考了解。

### 范围
无论是年报还是季报，都至少包括三张报表：资产负债表、利润表、现金流量表。
