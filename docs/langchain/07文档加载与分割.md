## 一、为什么需要文档处理

RAG（检索增强生成）的第一步是把文档变成可检索的片段：

```mermaid
flowchart LR
    A[原始文档] --> B[Document Loader]
    B --> C[Document 对象]
    C --> D[Text Splitter]
    D --> E[文档分块]
    E --> F[向量化 + 存储]
```

## 二、Document Loader

LangChain 提供几十种 Loader，加载不同格式的文档：

### 2.1 文本文件

```python
from langchain_community.document_loaders import TextLoader

loader = TextLoader("./data/faq.txt", encoding="utf-8")
docs = loader.load()
# [Document(page_content="...", metadata={"source": "./data/faq.txt"})]
```

### 2.2 CSV

```python
from langchain_community.document_loaders import CSVLoader

loader = CSVLoader("./data/products.csv", content_columns=["name", "description"])
docs = loader.load()
# 每行一个 Document
```

### 2.3 PDF

```python
from langchain_community.document_loaders import PyPDFLoader

loader = PyPDFLoader("./data/product_manual.pdf")
docs = loader.load()
# 每页一个 Document，metadata 包含页码
for doc in docs:
    print(f"第 {doc.metadata['page']} 页: {doc.page_content[:100]}")
```

### 2.4 Web 页面

```python
from langchain_community.document_loaders import WebBaseLoader

loader = WebBaseLoader("https://example.com/faq")
docs = loader.load()
```

### 2.5 目录批量加载

```python
from langchain_community.document_loaders import DirectoryLoader

loader = DirectoryLoader(
    "./data/",
    glob="**/*.md",          # 匹配所有 md 文件
    loader_cls=TextLoader,
    loader_kwargs={"encoding": "utf-8"},
    show_progress=True,
)
docs = loader.load()
```

### 2.6 Document 对象

所有 Loader 返回的都是 `Document` 对象：

```python
from langchain_core.documents import Document

doc = Document(
    page_content="商品内容...",
    metadata={
        "source": "products.csv",
        "row": 0,
    },
)

# 自定义元数据（后续检索时可以过滤）
doc.metadata["category"] = "电子产品"
```

## 三、Text Splitter

文档太长，需要分块才能有效检索：

### 3.1 RecursiveCharacterTextSplitter（推荐）

递归分割，优先在段落、句子边界切分：

```python
from langchain_text_splitters import RecursiveCharacterTextSplitter

splitter = RecursiveCharacterTextSplitter(
    chunk_size=500,       # 每块最大字符数
    chunk_overlap=50,     # 块之间重叠字符数
    separators=["\n\n", "\n", "。", "！", "？", "，", " ", ""],  # 分割优先级
)

chunks = splitter.split_documents(docs)
print(f"原始文档: {len(docs)} 个, 分块后: {len(chunks)} 个")
```

### 3.2 分块参数选择

| 参数 | 建议值 | 说明 |
|------|--------|------|
| `chunk_size` | 300-1000 | 太小丢失上下文，太大检索不精确 |
| `chunk_overlap` | chunk_size 的 10% | 保证边界信息不丢失 |

**客服场景建议**：FAQ 类文档用 300-500，产品手册用 500-1000。

### 3.3 按语义分割

```python
from langchain_text_splitters import MarkdownHeaderTextSplitter

# 先按 Markdown 标题分割
md_splitter = MarkdownHeaderTextSplitter(
    headers_to_split_on=[
        ("#", "h1"),
        ("##", "h2"),
        ("###", "h3"),
    ]
)
md_chunks = md_splitter.split_text(markdown_text)

# 再用 Recursive 分割长段落
char_splitter = RecursiveCharacterTextSplitter(chunk_size=500, chunk_overlap=50)
final_chunks = char_splitter.split_documents(md_chunks)
```

### 3.4 代码分割

```python
from langchain_text_splitters import Language, RecursiveCharacterTextSplitter

splitter = RecursiveCharacterTextSplitter.from_language(
    language=Language.PYTHON,
    chunk_size=1000,
    chunk_overlap=100,
)
```

## 四、客服知识库实战

```python
from langchain_community.document_loaders import DirectoryLoader, TextLoader
from langchain_text_splitters import RecursiveCharacterTextSplitter

def load_knowledge_base(data_dir: str = "./data") -> list:
    """加载并分割知识库文档"""
    # 1. 加载所有文档
    loader = DirectoryLoader(
        data_dir,
        glob="**/*.{txt,md}",
        loader_cls=TextLoader,
        loader_kwargs={"encoding": "utf-8"},
        show_progress=True,
    )
    docs = loader.load()

    # 2. 添加元数据
    for doc in docs:
        # 从文件路径提取分类
        path = doc.metadata["source"]
        if "faq" in path:
            doc.metadata["category"] = "faq"
        elif "product" in path:
            doc.metadata["category"] = "product"
        elif "policy" in path:
            doc.metadata["category"] = "policy"

    # 3. 分割
    splitter = RecursiveCharacterTextSplitter(
        chunk_size=500,
        chunk_overlap=50,
        separators=["\n\n", "\n", "。", "！", "？", "，", " ", ""],
    )
    chunks = splitter.split_documents(docs)

    print(f"加载 {len(docs)} 个文档，分割为 {len(chunks)} 个块")
    return chunks

# 使用
chunks = load_knowledge_base()
```

## 五、小结

| 组件 | 用途 |
|------|------|
| Document Loader | 加载各类格式文档 |
| Document | 统一文档对象（content + metadata） |
| RecursiveCharacterTextSplitter | 通用文本分割（推荐） |
| MarkdownHeaderTextSplitter | 按标题结构分割 |
| chunk_size / chunk_overlap | 分块参数调优 |
