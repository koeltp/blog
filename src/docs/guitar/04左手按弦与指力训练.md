## 一、品格的原理

### 1.1 什么是品格

指板上的金属横条叫"品丝"，两根品丝之间的区域叫"品格"。每按高一品，音高升高一个"半音"。

```
琴头 →
 |  |  |  |  |  |
 1  2  3  4  5  6   ← 品格编号
 ↑
 琴枕（不算品格，但按在琴枕与第1品之间 = 空弦音）
```

### 1.2 为什么按弦能改变音高

```mermaid
flowchart LR
    A[手指按在某品] --> B[弦的有效振动长度缩短]
    B --> C[振动频率升高]
    C --> D[音高升高]
```

- 空弦（不按）→ 弦全长振动 → 最低音
- 按第 1 品 → 弦从第 1 品丝开始振动 → 升高半音
- 按第 5 品 → 弦长缩短约 1/4 → 升高 5 个半音（纯四度）

> **关键**：你按的不是品丝本身，而是**品丝前方**的弦。手指放在两品之间、靠近琴身方向那根品丝的位置。

## 二、按弦的精确位置

### 2.1 最佳按弦点

```
琴头方向        琴身方向
  ←
  |---品丝---|---品丝---|
       ①          ②
   太靠后       最佳
              （靠近品丝，但不超过）
```

| 位置 | 效果 |
|------|------|
| ① 太靠后（远离品丝） | 需要极大力度，音准偏低 |
| ② 靠近品丝（推荐） | 省力、音准好、音色亮 |
| 超过品丝 | 闷音、发死 |

### 2.2 按弦力度

按弦力度有个"刚好"的临界点：

- **不够** → 弦没贴住品丝 → 打品、嗡嗡声
- **刚好** → 弦刚好贴住品丝 → 干净、省力
- **过头** → 弦过度拉伸 → 音偏高、手指疲劳

> **练习方法**：从最轻的力度开始，逐渐加大，找到"刚好能弹响"的那个力度，那就是最佳力度。

## 三、手指的独立性

吉他的难点在于：**四个手指要能独立、快速、精确地运动**。

### 3.1 手指编号

| 编号 | 手指 |
|------|------|
| 1 | 食指 |
| 2 | 中指 |
| 3 | 无名指 |
| 4 | 小指 |

### 3.2 独立性问题

| 现象 | 原因 |
|------|------|
| 无名指按弦时中指跟着动 | 指间肌腱相连，需训练独立 |
| 小指按不下去 | 力量弱，需要专门训练 |
| 按第 4 指时第 1 指抬起 | 手指联动 |

> **真相**：无名指和小指共享部分肌腱，这是生理结构决定的。通过练习能改善，但不能完全消除联动。

## 四、爬格子练习（核心）

爬格子是吉他手一辈子的基本功，训练手指力量、独立性、扩张度、节奏感。

### 4.1 基础爬格子（1-2-3-4）

在第 1 弦上，食指按 1 品、中指 2 品、无名指 3 品、小指 4 品，依次拨响：

```
第1弦  1品  2品  3品  4品
       1    2    3    4
       ●    ●    ●    ●
       ↓    ↓    ↓    ↓
       拨   拨   拨   拨
```

然后换到第 2 弦重复，一直到第 6 弦，再从第 6 弦倒回来。

```
1弦: 1 2 3 4
2弦: 1 2 3 4
3弦: 1 2 3 4
4弦: 1 2 3 4
5弦: 1 2 3 4
6弦: 1 2 3 4
```

**要求：**
- 每个音清晰、无杂音
- 按弦手指**留在原位**，不要抬起来（按下一个手指时，前一个手指保持按住）
- 节奏均匀（建议配节拍器，60-80 BPM，一拍一音）

### 4.2 变体训练

#### 变体 1：1-3-2-4（扩张训练）

```
1弦: 1 3 2 4  ← 食指1品，无名指3品（跨一个品），中指2品，小指4品
```

训练手指扩张度。

#### 变体 2：1-4-2-3（独立性训练）

```
1弦: 1 4 2 3  ← 食指1品，小指4品（跨度大），中指2品，无名指3品
```

打乱顺序，训练大脑对手指的控制。

#### 变体 3：横向爬格（换弦不换品）

```
1品: 6弦1 5弦1 4弦1 3弦1 2弦1 1弦1
2品: 6弦2 5弦2 4弦2 3弦2 2弦2 1弦2
3品: ...
```

训练换弦的灵活性。

### 4.3 进阶：跨弦练习

跳过一根弦拨奏，训练右手精度：

```
1弦: 1 2 3 4
3弦: 1 2 3 4  ← 跳过2弦
2弦: 1 2 3 4
4弦: 1 2 3 4  ← 跳过3弦
```

## 五、扩张度训练

新手最大的生理障碍是手指打不开。以下练习能逐步改善：

### 5.1 蝴蝶练习

食指按 1 品、小指按 4 品，**同时按下并保持**，然后中指尝试按 2 品、无名指按 3 品：

```
1品  2品  3品  4品
 ●    ●    ●    ●
 1    2    3    4
(食) (中) (无) (小)
```

如果按不到 3-4 品，先按 1-3 品、2-4 品分开练，再合起来。

### 5.2 逐品扩张

从第 5 品开始（品格间距小，容易按），练熟后逐渐移向第 1 品（间距最大）：

```
5品: 1 2 3 4  ← 容易
3品: 1 2 3 4  ← 中等
1品: 1 2 3 4  ← 最难
```

## 六、本章练习计划

### 每日 15 分钟基本功

| 时间 | 内容 | 目的 |
|------|------|------|
| 3 分钟 | 基础爬格子 1-2-3-4（6弦→1弦→6弦） | 热身 |
| 3 分钟 | 变体 1-3-2-4 | 扩张 |
| 3 分钟 | 变体 1-4-2-3 | 独立性 |
| 3 分钟 | 跨弦爬格 | 右手精度 |
| 3 分钟 | 蝴蝶练习 | 扩张度 |

> **节拍器是必需品**。从 60 BPM 开始，能稳定弹 1 分钟不乱就 +5 BPM，目标 120 BPM。

## 七、常见误区与 FAQ

| 问题 | 原因 | 解决 |
|------|------|------|
| 小指按不下去、发抖 | 力量弱 | 专门练 1-4 跨度，慢慢来 |
| 按弦后手指粘住弦 | 指腹出汗 | 洗手擦干，必要时用婴儿爽身粉 |
| 第 4 指总翘起来 | 联动 | 练习时强制压住，培养控制力 |
| 爬格子速度上不去 | 节奏不稳 | 回到慢速，用节拍器，先准再快 |
| 手腕酸痛 | 姿势错误 | 检查手腕是否外翻过度，适当放松 |

---

## 小结

- **品格原理**：每按高一品，升半音
- **按弦位置**：靠近品丝，不超过
- **按弦力度**：刚好能弹响即可
- **爬格子**：1-2-3-4 基础 + 变体训练独立性
- **扩张**：从 5 品向 1 品逐步挑战
- **节拍器**：从 60 BPM 起步，目标 120 BPM
