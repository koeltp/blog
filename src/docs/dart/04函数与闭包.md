# 函数与闭包

> 上一章我们实现了 Todo 应用的命令解析，但所有逻辑都堆在 main() 里。本章学习函数的各种写法，把代码重构为清晰的函数式架构。

Dart 中函数是一等公民，可以赋值给变量、作为参数传递、作为返回值。

## 函数声明

### 基本形式

```dart
int add(int a, int b) {
  return a + b;
}

void greet(String name) {
  print('Hello, $name!');
}
```

### 箭头函数

函数体只有一个表达式时，可以用 `=>` 简写：

```dart
int add(int a, int b) => a + b;
void greet(String name) => print('Hello, $name!');
```

## 参数

### 位置参数

```dart
void sayHello(String name, int age) {
  print('$name, $age 岁');
}

sayHello('Dart', 10);  // 必须按顺序传
```

### 可选位置参数

用 `[]` 包裹，必须放在参数列表末尾：

```dart
void sayHello(String name, [int? age]) {
  print('$name${age != null ? ', $age 岁' : ''}');
}

sayHello('Dart');       // Dart
sayHello('Dart', 10);  // Dart, 10 岁
```

### 命名参数

用 `{}` 包裹，调用时用 `name: value`：

```dart
void createUser({required String name, int age = 0}) {
  print('name: $name, age: $age');
}

createUser(name: 'Dart', age: 10);  // name: Dart, age: 10
createUser(name: 'Flutter');         // name: Flutter, age: 0
// createUser();                     // ❌ name 是 required
```

> Flutter 中命名参数非常常见，几乎所有 Widget 构造函数都用命名参数。

### 默认值

```dart
void connect(String host, {int port = 3306, String protocol = 'tcp'}) {
  print('$protocol://$host:$port');
}

connect('localhost');                      // tcp://localhost:3306
connect('localhost', port: 5432);          // tcp://localhost:5432
connect('localhost', protocol: 'ws');      // ws://localhost:3306
```

### required 关键字

```dart
void sendEmail({required String to, required String subject, String body = ''}) {
  print('To: $to, Subject: $subject');
}

sendEmail(to: 'a@b.com', subject: 'Hello');  // ✅
// sendEmail(to: 'a@b.com');                  // ❌ subject 是 required
```

## 匿名函数

```dart
// 基本形式
var multiply = (int a, int b) {
  return a * b;
};
print(multiply(3, 4));  // 12

// 箭头简写
var square = (int n) => n * n;
print(square(5));  // 25
```

## 闭包

闭包是能访问外部作用域变量的函数，即使外部函数已返回：

```dart
Function makeCounter() {
  int count = 0;
  return () {
    count++;
    return count;
  };
}

var counter = makeCounter();
print(counter());  // 1
print(counter());  // 2
print(counter());  // 3

// 每次调用 makeCounter() 创建独立的闭包
var counter2 = makeCounter();
print(counter2());  // 1（独立计数）
```

闭包的实际应用 — 工厂函数：

```dart
Function multiplier(int factor) {
  return (int value) => value * factor;
}

var double = multiplier(2);
var triple = multiplier(3);

print(double(5));  // 10
print(triple(5));  // 15
```

## 高阶函数

接受函数作为参数或返回函数的函数：

```dart
// 接受函数参数
void performOperation(int a, int b, int Function(int, int) operation) {
  print('结果：${operation(a, b)}');
}

performOperation(5, 3, (a, b) => a + b);  // 结果：8
performOperation(5, 3, (a, b) => a * b);  // 结果：15
```

### 集合的常用高阶函数

```dart
var numbers = [1, 2, 3, 4, 5];

// map — 映射
var doubled = numbers.map((n) => n * 2).toList();
print(doubled);  // [2, 4, 6, 8, 10]

// where — 过滤
var evens = numbers.where((n) => n.isEven).toList();
print(evens);    // [2, 4]

// reduce — 归约
var sum = numbers.reduce((a, b) => a + b);
print(sum);      // 15

// fold — 带初始值的归约
var product = numbers.fold(1, (a, b) => a * b);
print(product);  // 120

// every — 全部满足
var allPositive = numbers.every((n) => n > 0);
print(allPositive);  // true

// any — 任一满足
var hasLarge = numbers.any((n) => n > 4);
print(hasLarge);     // true

// sort — 排序（原地修改）
var sorted = [3, 1, 4, 1, 5];
sorted.sort((a, b) => a.compareTo(b));
print(sorted);  // [1, 1, 3, 4, 5]

// forEach — 遍历
numbers.forEach((n) => print(n));
```

## typedef — 函数类型别名

```dart
typedef Calculator = int Function(int, int);

void performCalc(int a, int b, Calculator calc) {
  print(calc(a, b));
}

performCalc(5, 3, (a, b) => a + b);   // 8
performCalc(5, 3, (a, b) => a * b);   // 15
```

## 小结

| 概念 | 关键点 |
|------|--------|
| 命名参数 | `{}` 包裹，Flutter 中最常用 |
| 可选位置参数 | `[]` 包裹，较少使用 |
| `required` | 命名参数必填标记 |
| 默认值 | `=` 指定默认值 |
| 箭头函数 | `=>` 单表达式简写 |
| 闭包 | 函数捕获外部变量 |

## 踩坑提示

::: warning 常见错误
1. **命名参数默认可选** — 忘记加 `required` 会导致参数可以不传，运行时拿到 null
2. **闭包捕获的是引用** — 循环中创建闭包可能捕获到循环变量的最终值
3. **`reduce` 空列表崩溃** — 对空列表调用 `reduce` 会抛出 StateError，用 `fold` 更安全
4. **位置参数在命名参数前面** — `void f(int a, {int b})` 合法，`void f({int b}, int a)` 不合法
:::

## 实战：重构为函数式架构

把第 3 章的命令解析重构为函数：

```dart
import 'dart:io';

// 命令处理函数
void handleAdd(String content) => print('添加待办：$content');
void handleList() => print('（列表功能将在第5章实现）');
void handleDone(int index) => print('完成第 $index 项');

// 解析并执行命令
void executeCommand(String input) {
  var parts = input.split(' ');
  var command = parts[0];

  switch (command) {
    case 'add':
      var content = parts.length > 1 ? parts.sublist(1).join(' ') : null;
      if (content != null && content.isNotEmpty) {
        handleAdd(content);
      } else {
        print('用法：add <内容>');
      }
    case 'list':
      handleList();
    case 'done':
      var index = parts.length > 1 ? int.tryParse(parts[1]) : null;
      if (index != null) {
        handleDone(index);
      } else {
        print('用法：done <编号>');
      }
    case 'quit':
      print('再见！');
      exit(0);
    default:
      print('未知命令：$command');
  }
}

void main() {
  print('=== Todo App ===');
  print('命令：add <内容> / list / done <编号> / quit');

  while (true) {
    stdout.write('\n> ');
    var input = stdin.readLineSync() ?? '';
    if (input.isNotEmpty) executeCommand(input);
  }
}
```
