# 运算符与控制流

> 上一章我们学会了声明变量和定义类型。本章学习运算符和控制流，然后用它们为 Todo 应用实现命令解析功能。

## 运算符

### 算术运算符

```dart
print(5 + 3);    // 8   加
print(5 - 3);    // 2   减
print(5 * 3);    // 15  乘
print(5 / 3);    // 1.6666...  除（结果是 double）
print(5 ~/ 3);   // 1   整除
print(5 % 3);    // 2   取模

// 自增/自减
var a = 5;
print(a++);      // 5（先返回再自增）
print(++a);      // 7（先自增再返回）
```

### 关系运算符

```dart
print(5 == 5);   // true   等于
print(5 != 3);   // true   不等于
print(5 > 3);    // true   大于
print(5 < 3);    // false  小于
print(5 >= 5);   // true   大于等于
print(5 <= 3);   // false  小于等于
```

### 逻辑运算符

```dart
bool a = true, b = false;
print(!a);       // false  非
print(a && b);   // false  与
print(a || b);   // true   或
```

### 赋值运算符

```dart
var a = 10;
a += 5;   // a = 15
a -= 3;   // a = 12
a *= 2;   // a = 24
a ~/= 5;  // a = 4

// ??= 空值赋值：仅当变量为 null 时赋值
int? b;
b ??= 42;   // b 为 42
b ??= 99;   // b 仍为 42
```

### 条件运算符

```dart
// 三目运算符
var age = 20;
var label = age >= 18 ? '成年' : '未成年';

// ?? 空值合并
String? name;
var display = name ?? '匿名';   // display = '匿名'
```

### 级联运算符

```dart
// .. 级联，在链式调用中返回原对象而非返回值
var paint = Paint()
  ..color = Colors.black
  ..strokeCap = StrokeCap.round
  ..strokeWidth = 5.0;

// 等价于
var paint = Paint();
paint.color = Colors.black;
paint.strokeCap = StrokeCap.round;
paint.strokeWidth = 5.0;
```

## 控制流

### if / else

```dart
var score = 85;

if (score >= 90) {
  print('优秀');
} else if (score >= 80) {
  print('良好');
} else if (score >= 60) {
  print('及格');
} else {
  print('不及格');
}
```

> 注意：条件必须是布尔值，`0`、`null`、空字符串都不是 `false`。

### switch / case

```dart
var day = 'Monday';

switch (day) {
  case 'Monday':
  case 'Tuesday':
  case 'Wednesday':
  case 'Thursday':
  case 'Friday':
    print('工作日');
    break;
  case 'Saturday':
  case 'Sunday':
    print('周末');
    break;
  default:
    print('未知');
}
```

Dart 3+ 的 switch 表达式：

```dart
var label = switch (day) {
  'Monday' || 'Tuesday' || 'Wednesday' || 'Thursday' || 'Friday' => '工作日',
  'Saturday' || 'Sunday' => '周末',
  _ => '未知',
};
```

### for 循环

```dart
// 标准 for
for (var i = 0; i < 5; i++) {
  print(i);
}

// for-in 遍历集合
var fruits = ['苹果', '香蕉', '橘子'];
for (var fruit in fruits) {
  print(fruit);
}

// forEach 方法
fruits.forEach((fruit) {
  print(fruit);
});
```

### while / do-while

```dart
// while：先判断再执行
var i = 0;
while (i < 5) {
  print(i);
  i++;
}

// do-while：先执行再判断
var j = 0;
do {
  print(j);
  j++;
} while (j < 5);
```

### break / continue

```dart
// break 跳出整个循环
for (var i = 0; i < 10; i++) {
  if (i == 5) break;
  print(i);  // 0, 1, 2, 3, 4
}

// continue 跳过当前迭代
for (var i = 0; i < 10; i++) {
  if (i % 2 == 0) continue;
  print(i);  // 1, 3, 5, 7, 9
}
```

### assert 断言

```dart
// 仅在调试模式生效，生产环境忽略
var age = 25;
assert(age >= 0, '年龄不能为负数');
assert(age < 150, '年龄不合理');
```

## 小结

- `~/` 是 Dart 特有的整除运算符，`/` 总是返回 `double`
- `??` 和 `??=` 是处理空值的利器
- `..` 级联运算符在 Flutter 中大量使用
- 条件必须是 `bool`，没有 JavaScript 那样的隐式转换
- Dart 3 的 switch 表达式更简洁

## 踩坑提示

::: warning 常见错误
1. **`/` 和 `~/` 混淆** — `10 / 3` 得 `3.333`，`10 ~/ 3` 得 `3`，需要整数时必须用 `~/`
2. **switch 忘记 break** — Dart 不支持 fall-through，每个 case 必须有 break/return/throw
3. **`==` 比较的是值** — Dart 的 `==` 默认比较引用，但 String 和基本类型比较的是值
4. **`||` 和 `&&` 短路** — `if (list != null && list.isNotEmpty)` 安全，反过来会报错
:::

## 实战：实现命令解析

用运算符和控制流为 Todo 应用添加命令解析：

```dart
import 'dart:io';

void main() {
  print('=== Todo App ===');
  print('命令：add <内容> / list / done <编号> / quit');

  while (true) {
    stdout.write('\n> ');
    var input = stdin.readLineSync() ?? '';

    if (input.isEmpty) continue;

    var parts = input.split(' ');
    var command = parts[0];

    switch (command) {
      case 'add':
        var content = parts.length > 1 ? parts.sublist(1).join(' ') : null;
        if (content != null && content.isNotEmpty) {
          print('添加待办：$content');
        } else {
          print('用法：add <内容>');
        }
      case 'list':
        print('（列表功能将在第5章实现）');
      case 'done':
        var index = parts.length > 1 ? int.tryParse(parts[1]) : null;
        if (index != null) {
          print('完成第 $index 项');
        } else {
          print('用法：done <编号>');
        }
      case 'quit':
        print('再见！');
        return;
      default:
        print('未知命令：$command');
    }
  }
}
```
