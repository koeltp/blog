# 变量、常量与类型系统

> 上一章我们搭建了开发环境并创建了 Todo 项目。本章学习 Dart 的变量声明和类型系统，然后用这些知识为 Todo 应用定义数据模型。

Dart 是强类型语言，但支持类型推断，写起来像动态语言一样灵活。

## 变量声明

### var — 类型推断

```dart
var name = 'Dart';     // 推断为 String
var year = 2026;       // 推断为 int
var pi = 3.14;         // 推断为 double
var isReady = true;    // 推断为 bool
```

一旦推断完成，类型不可更改：

```dart
var name = 'Dart';
name = 'Flutter';      // ✅ 同类型可以重新赋值
// name = 42;          // ❌ 编译错误，不能赋 int 给 String
```

### 显式类型

```dart
String name = 'Dart';
int count = 10;
double price = 9.99;
bool isValid = true;
```

### dynamic — 动态类型

```dart
dynamic value = 'hello';
value = 42;            // ✅ 可以改变类型
value = true;          // ✅ 任意类型都行
```

> 尽量避免使用 `dynamic`，它会绕过类型检查。优先用 `Object?` 或泛型。

### Object — 所有类型的基类

```dart
Object value = 'hello';
value = 42;            // ✅ Object 允许赋任何值
```

## 常量声明

### final — 运行时常量

```dart
final String name = 'Dart';
final age = 10;        // 类型推断

// final 只能赋值一次
// name = 'Flutter';  // ❌ 编译错误
```

`final` 可以在运行时确定值：

```dart
final now = DateTime.now();  // ✅ 运行时计算
```

### const — 编译时常量

```dart
const pi = 3.14159;
const maxRetry = 3;

// const 必须在编译期就能确定值
// const time = DateTime.now();  // ❌ 运行时才能计算
```

### final vs const

| 特性 | final | const |
|------|-------|-------|
| 赋值次数 | 一次 | 一次 |
| 确定时机 | 运行时 | 编译时 |
| 对象内容 | 可变（如 List 可添加元素） | 不可变（深不可变） |
| 典型场景 | 运行时配置、单例 | 枚举值、配置常量 |

```dart
final list1 = [1, 2, 3];
list1.add(4);          // ✅ final List 内容可变

const list2 = [1, 2, 3];
// list2.add(4);       // ❌ const List 内容不可变
```

## 内置类型

### 数字：int / double

```dart
int count = 42;
double price = 9.99;

// 整数除法用 ~/
int result = 10 ~/ 3;  // 3

// 类型转换
int a = 42;
double b = a.toDouble();   // 42.0
int c = 3.14.toInt();      // 3

// 常用方法
print(42.isEven);          // true
print(42.isOdd);           // false
print((-5).abs());         // 5
print(3.14.round());       // 3
print(3.14.toStringAsFixed(1));  // "3.1"
```

### 字符串：String

```dart
var s1 = '单引号';
var s2 = "双引号";
var s3 = '''多行
字符串''';

// 字符串插值
var name = 'Dart';
print('Hello, $name!');           // Hello, Dart!
print('长度：${name.length}');     // 长度：4
print('2 + 3 = ${2 + 3}');       // 2 + 3 = 5

// 常用方法
print('hello'.toUpperCase());     // HELLO
print('  hello  '.trim());        // hello
print('a,b,c'.split(','));       // [a, b, c]
print('Dart'.contains('art'));   // true
print('hello'.substring(0, 3));  // hel
```

### 布尔：bool

```dart
bool isValid = true;
bool isEmpty = false;

// Dart 中只有 true 和 false 是布尔值
// 以下在 JavaScript 中可行，但在 Dart 中不行：
// var name = 'Dart';
// if (name) { ... }    // ❌ 编译错误，String 不是 bool
// if (name.isNotEmpty) { ... }  // ✅ 正确做法
```

### 符文：Runes

用于表示 Unicode 字符：

```dart
var heart = '\u2764';           // ❤
var flag = '\u{1F1E8}\u{1F1F3}'; // 🇨🇳
print(heart);
print(flag);
```

## 类型检查与转换

### is / is!

```dart
var value = 'hello';
if (value is String) {
  print(value.length);  // 自动类型推导为 String
}

if (value is! int) {
  print('不是整数');
}
```

### as — 类型转换

```dart
Object value = 'hello';
String str = value as String;  // 强制转换

// 如果类型不匹配会抛出异常
// int num = value as int;     // ❌ 运行时 TypeError
```

## 小结

| 声明方式 | 类型 | 可变性 | 使用场景 |
|---------|------|--------|---------|
| `var` | 推断 | 可重新赋值 | 局部变量首选 |
| `显式类型` | 指定 | 可重新赋值 | 需要明确类型时 |
| `final` | 推断/指定 | 不可重新赋值 | 运行时常量 |
| `const` | 推断/指定 | 编译期深不可变 | 编译时常量 |
| `dynamic` | 动态 | 任意 | 避免使用，优先泛型 |

## 踩坑提示

::: warning 常见错误
1. **`var` 不是动态类型** — `var x = 'hi'; x = 42;` 会报错，var 推断后类型固定
2. **`final` 的集合内容可变** — `final list = [1,2]; list.add(3);` 是合法的，final 只限制引用不变
3. **`const` 构造函数要求所有字段 final** — 类中有非 final 字段就不能用 const 构造函数
4. **`/` 总是返回 double** — `10 / 3` 是 `3.333...`，整除用 `~/`
5. **布尔值没有隐式转换** — `if (name)` 编译错误，必须 `if (name.isNotEmpty)`
:::

## 实战：定义 Todo 数据模型

用本章学到的变量和类型知识，为 Todo 应用定义数据：

```dart
// bin/todo_app.dart
void main(List<String> arguments) {
  // 用 var 声明局部变量
  var appName = 'Todo App';
  final version = '1.0.0';

  // Todo 项的数据（暂时用简单变量，后续章节会用类来封装）
  String title = '学习 Dart';
  bool completed = false;
  int priority = 1;  // 1=高, 2=中, 3=低
  String category = '学习';

  print('=== $appName v$version ===');
  print('待办：$title');
  print('状态：${completed ? "已完成" : "未完成"}');
  print('优先级：$priority | 分类：$category');
}
```
