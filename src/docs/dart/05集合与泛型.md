# 集合与泛型

> 上一章我们用函数重构了 Todo 应用的架构。本章学习 List/Set/Map，让 Todo 应用真正能存储和管理待办列表。

## List — 列表

### 创建

```dart
// 字面量
var fruits = ['苹果', '香蕉', '橘子'];

// 指定类型
List<int> numbers = [1, 2, 3];

// 固定长度
var fixed = List.filled(5, 0);  // [0, 0, 0, 0, 0]

// 生成器
var range = List.generate(5, (i) => i * 2);  // [0, 2, 4, 6, 8]

// 空列表
var empty = <String>[];  // 显式泛型
```

### 常用操作

```dart
var fruits = ['苹果', '香蕉', '橘子'];

// 访问
print(fruits[0]);           // 苹果
print(fruits.first);        // 苹果
print(fruits.last);         // 橘子
print(fruits.length);       // 3

// 添加
fruits.add('葡萄');         // 末尾添加
fruits.insert(1, '芒果');   // 指定位置插入
fruits.addAll(['梨', '桃']); // 批量添加

// 删除
fruits.remove('香蕉');      // 按值删除
fruits.removeAt(0);         // 按索引删除
fruits.removeLast();        // 删除最后一个

// 查找
print(fruits.contains('苹果'));     // true
print(fruits.indexOf('橘子'));      // 索引
print(fruits.where((f) => f.length > 1).toList());  // 过滤

// 排序
var nums = [3, 1, 4, 1, 5];
nums.sort();                // [1, 1, 3, 4, 5]
nums.sort((a, b) => b.compareTo(a));  // 降序

// 截取
var sub = nums.sublist(1, 3);  // [1, 3]

// 遍历
for (var fruit in fruits) {
  print(fruit);
}
fruits.forEach((fruit) => print(fruit));
```

### 展开运算符

```dart
var a = [1, 2, 3];
var b = [4, 5, 6];

// 拼接
var c = [...a, ...b];           // [1, 2, 3, 4, 5, 6]

// 空安全展开
List<int>? nullable = null;
var d = [...a, ...?nullable];   // [1, 2, 3]（null 时跳过）
```

## Set — 集合

元素唯一，无序。

```dart
// 创建
var colors = <String>{'红', '绿', '蓝'};
var empty = <int>{};           // 空集合（注意：{} 是 Map 不是 Set）

// 添加/删除
colors.add('黄');
colors.remove('绿');

// 判断
print(colors.contains('红'));   // true
print(colors.length);           // 3

// 集合运算
var a = {1, 2, 3, 4};
var b = {3, 4, 5, 6};

print(a.intersection(b));  // {3, 4}    交集
print(a.union(b));         // {1,2,3,4,5,6} 并集
print(a.difference(b));    // {1, 2}    差集（a有b没有）
```

## Map — 映射

键值对集合。

```dart
// 创建
var person = {
  'name': 'Dart',
  'age': 10,
};

// 指定类型
Map<String, int> scores = {
  'Alice': 95,
  'Bob': 87,
};

// 访问
print(person['name']);          // Dart
print(person['height']);        // null（键不存在返回 null）
print(person.containsKey('name'));  // true

// 添加/修改
person['email'] = 'dart@dev';  // 新增
person['age'] = 11;            // 修改

// 删除
person.remove('email');

// 遍历
person.forEach((key, value) {
  print('$key: $value');
});

for (var entry in person.entries) {
  print('${entry.key} = ${entry.value}');
}

// 常用属性
print(person.keys);     // 所有键
print(person.values);   // 所有值
print(person.length);   // 键值对数量
print(person.isEmpty);  // 是否为空
```

### fromEntries / toMap

```dart
var scores = {'a': 3, 'b': 1, 'c': 2};

// 按 value 排序生成新 Map
var sorted = Map.fromEntries(
  scores.entries.toList()..sort((a, b) => a.value.compareTo(b.value))
);
print(sorted);  // {b: 1, c: 2, a: 3}

// toMap 转换
var doubled = scores.map((key, value) => MapEntry(key, value * 2));
print(doubled);  // {a: 6, b: 2, c: 4}
```

## 泛型

### 基本用法

```dart
// 泛型类
class Box<T> {
  T value;
  Box(this.value);

  T getValue() => value;
}

var intBox = Box<int>(42);
var strBox = Box<String>('hello');

// 泛型方法
T first<T>(List<T> items) {
  return items.first;
}

print(first<int>([1, 2, 3]));     // 1
print(first<String>(['a', 'b'])); // a
```

### 泛型约束

```dart
// 限制 T 必须是 num 的子类
T max<T extends num>(T a, T b) {
  return a >= b ? a : b;
}

print(max(3, 5));       // 5
print(max(3.14, 2.71)); // 3.14
// max('a', 'b');       // ❌ String 不是 num 子类
```

### 泛型集合的类型安全

```dart
// 没有泛型，类型不安全
var list = [1, 'two', 3.0];  // List<Object>

// 有泛型，类型安全
var ints = <int>[1, 2, 3];
// ints.add('four');          // ❌ 编译错误
```

## 集合的常用技巧

### 去重

```dart
var items = [1, 2, 2, 3, 3, 3];
var unique = items.toSet().toList();  // [1, 2, 3]
```

### 分组

```dart
var words = ['apple', 'bat', 'cat', 'dog', 'ant'];
var grouped = <int, List<String>>{};

for (var word in words) {
  var len = word.length;
  grouped.putIfAbsent(len, () => []).add(word);
}
print(grouped);  // {5: [apple], 3: [bat, cat, dog, ant]}
```

### 展平嵌套列表

```dart
var nested = [[1, 2], [3, 4], [5]];
var flat = nested.expand((list) => list).toList();
print(flat);  // [1, 2, 3, 4, 5]
```

## 小结

| 集合 | 特点 | 典型场景 |
|------|------|---------|
| `List` | 有序、可重复 | 数据序列、有序遍历 |
| `Set` | 无序、唯一 | 去重、集合运算 |
| `Map` | 键值对 | 配置、查找表、分组 |

- 优先使用字面量语法创建集合：`[]`、`{}`、`{}`
- 泛型保证类型安全，Flutter 中到处是泛型

## 踩坑提示

::: warning 常见错误
1. **`{}` 是 Map 不是 Set** — `var x = {}` 创建的是 Map，空 Set 要写成 `<int>{}`
2. **`List.remove` 返回 bool** — `remove` 返回是否删除成功，不是被删除的元素
3. **`sort` 是原地修改** — `list.sort()` 会修改原列表，想保留原列表先 `list.toList()..sort()`
4. **`Map` 的键顺序** — Dart 的 Map 保持插入顺序，但不要依赖它做逻辑判断
5. **`expand` vs `map`** — `map` 一对一，`expand` 一对多（展平嵌套）
:::

## 实战：用集合管理 Todo 列表

用 List 和 Map 实现真正的 Todo 列表管理：

```dart
import 'dart:io';

// 用 Map 表示每个 Todo 项
List<Map<String, dynamic>> todos = [];
int nextId = 1;

void handleAdd(String content) {
  todos.add({
    'id': nextId++,
    'content': content,
    'completed': false,
    'createdAt': DateTime.now().toIso8601String(),
  });
  print('已添加：$content (#$${nextId - 1})');
}

void handleList() {
  if (todos.isEmpty) {
    print('没有待办事项');
    return;
  }
  // 用 where 过滤 + map 格式化
  var pending = todos.where((t) => !t['completed']);
  var completed = todos.where((t) => t['completed']);

  if (pending.isNotEmpty) {
    print('\n待完成：');
    pending.forEach((t) => print('  #${t['id']} ${t['content']}'));
  }
  if (completed.isNotEmpty) {
    print('\n已完成：');
    completed.forEach((t) => print('  #${t['id']} ${t['content']} ✓'));
  }
  print('\n共 ${todos.length} 项，${completed.length} 项已完成');
}

void handleDone(int id) {
  var todo = todos.where((t) => t['id'] == id).firstOrNull;
  if (todo != null) {
    todo['completed'] = true;
    print('已完成：${todo['content']}');
  } else {
    print('找不到 #$id');
  }
}
```
