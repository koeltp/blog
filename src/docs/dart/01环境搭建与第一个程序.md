# 环境搭建与第一个程序

> 欢迎来到 Dart 教程！本章从零开始，带你安装开发环境、创建项目、运行第一个程序。这是整个教程的基础，后续所有章节都依赖本章搭建的环境。

## 安装 Dart SDK

### Windows

推荐使用 Chocolatey：

```bash
choco install dart-sdk
```

或手动下载：访问 [dart.dev/get-dart](https://dart.dev/get-dart)，下载安装包并配置环境变量。

### macOS

```bash
brew tap dart-lang/dart
brew install dart
```

### Linux

```bash
sudo apt-get update
sudo apt-get install apt-transport-https
sudo sh -c 'wget -qO- https://dl-ssl.google.com/linux/linux_signing_key.pub | apt-key add -'
sudo sh -c 'wget -qO- https://storage.googleapis.com/download.dartlang.org/linux/debian/dart_stable.list > /etc/apt/sources.list.d/dart_stable.list'
sudo apt-get update
sudo apt-get install dart
```

### 验证安装

```bash
dart --version
# Dart SDK version: 3.x.x
```

## 开发工具

### VS Code（推荐）

1. 安装 [Dart 扩展](https://marketplace.visualstudio.com/items?itemName=Dart-Code.dart-code)
2. 安装 [Flutter 扩展](https://marketplace.visualstudio.com/items?itemName=Dart-Code.flutter)（如需 Flutter 开发）
3. 扩展提供：语法高亮、代码补全、调试、格式化

### IntelliJ / Android Studio

安装 Dart 和 Flutter 插件即可。

### DartPad

在线运行 Dart 代码，无需安装：[dartpad.dev](https://dartpad.dev/)

## 第一个程序

创建文件 `hello.dart`：

```dart
void main() {
  print('Hello, Dart!');
}
```

运行：

```bash
dart run hello.dart
# Hello, Dart!
```

## 程序结构解析

```dart
// 这是单行注释

/*
  这是多行注释
*/

/// 这是文档注释，用于生成 API 文档

// main() 是程序入口，每个 Dart 程序必须有
void main() {
  // print() 是最基础的输出函数
  print('Hello, Dart!');

  // 支持字符串插值
  var name = 'World';
  print('Hello, $name!');

  // 表达式插值用 ${}
  print('2 + 3 = ${2 + 3}');
}
```

## 交互式输入

```dart
import 'dart:io';

void main() {
  stdout.write('请输入你的名字：');
  var name = stdin.readLineSync();
  print('你好，$name！');
}
```

## 命令行工具

Dart SDK 自带几个常用命令：

| 命令 | 用途 |
|------|------|
| `dart run` | 运行 Dart 程序 |
| `dart compile exe` | 编译为可执行文件 |
| `dart compile js` | 编译为 JavaScript |
| `dart format` | 格式化代码 |
| `dart analyze` | 静态分析代码 |
| `dart test` | 运行测试 |
| `dart create` | 创建新项目 |

## 创建项目

```bash
# 创建控制台应用
dart create -t console my_app

# 创建包
dart create -t package my_package

# 项目结构
# my_app/
# ├── bin/
# │   └── my_app.dart    # 入口文件
# ├── lib/
# │   └── my_app.dart    # 库代码
# ├── test/
# │   └── my_app_test.dart
# ├── pubspec.yaml        # 依赖配置
# └── analysis_options.yaml
```

## 小结

- Dart SDK 安装简单，Windows/macOS/Linux 均有包管理器支持
- VS Code + Dart 扩展是最佳开发组合
- `main()` 是程序入口，`print()` 是基础输出
- `dart create` 可以快速创建项目脚手架

## 踩坑提示

::: warning 常见问题
1. **`dart` 命令找不到** — 安装后重启终端，或检查环境变量是否包含 Dart SDK 的 `bin` 目录
2. **VS Code 无代码提示** — 确认已安装 Dart 扩展，并打开的是 `pubspec.yaml` 所在的目录
3. **`dart create` 失败** — 确认网络正常，Dart 需要从 pub.dev 下载依赖
4. **中文输出乱码** — Windows 终端执行 `chcp 65001` 切换到 UTF-8 编码
:::

## 实战：创建 Todo 项目

从本章开始，我们将逐步构建一个命令行 Todo 应用。先创建项目骨架：

```bash
dart create -t console todo_app
cd todo_app
```

修改 `bin/todo_app.dart`：

```dart
void main(List<String> arguments) {
  print('=== Todo App ===');
  print('命令：add / list / done / quit');

  // 后续章节会逐步完善这个程序
  print('（功能开发中...）');
}
```

运行验证：

```bash
dart run bin/todo_app.dart
# === Todo App ===
# 命令：add / list / done / quit
# （功能开发中...）
```

## 练习题

1. 安装 Dart SDK，运行 `dart --version` 确认版本 >= 3.0
2. 用 `dart create` 创建一个控制台项目，修改 `main()` 输出你的名字
3. 编写一个程序，读取用户输入的数字并输出其平方值
4. 尝试 `dart format` 和 `dart analyze` 命令，理解它们的作用
