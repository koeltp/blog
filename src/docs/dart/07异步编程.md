# 异步编程

> 上一章我们用类封装了 Todo 应用，但数据只存在内存中，重启就丢失。本章学习异步编程，实现文件持久化。

异步编程是 Dart 的核心能力，也是 Flutter 框架的基础。理解 Future 和 Stream 是写好 Dart 代码的关键。

## 事件循环

Dart 是单线程模型，通过事件循环（Event Loop）处理异步操作：

```
主线程执行同步代码
    ↓
事件循环检查事件队列
    ↓
有事件？→ 取出执行 → 回到检查
没有？→ 等待新事件
```

```
┌─────────────────────────────┐
│         事件循环             │
│  ┌───────────────────────┐  │
│  │  微任务队列(MicroTask) │  │  ← 优先级高
│  └───────────────────────┘  │
│  ┌───────────────────────┐  │
│  │  事件队列(Event)       │  │  ← 优先级低
│  └───────────────────────┘  │
└─────────────────────────────┘
```

- **微任务队列**：`scheduleMicrotask()` 产生，优先执行
- **事件队列**：I/O、定时器、用户交互等产生

## Future

Future 表示一个异步操作的最终结果，类似 JavaScript 的 Promise。

### 创建 Future

```dart
// 立即完成
Future<int> immediate() {
  return Future.value(42);
}

// 延迟完成
Future<String> delayed() {
  return Future.delayed(Duration(seconds: 2), () => '完成');
}

// 模拟异步操作
Future<String> fetchUser() {
  return Future(() {
    // 模拟网络请求
    return '用户数据';
  });
}
```

### then / catchError

```dart
fetchUser()
  .then((data) {
    print('获取到：$data');
    return data.length;
  })
  .then((length) {
    print('数据长度：$length');
  })
  .catchError((error) {
    print('出错了：$error');
  });
```

### async / await

更直观的异步写法，代码看起来像同步：

```dart
Future<String> fetchUser() async {
  // await 暂停执行，等 Future 完成后继续
  var data = await Future.delayed(Duration(seconds: 1), () => '用户数据');
  print('获取到：$data');
  return data;
}
```

### 错误处理

```dart
Future<void> loadData() async {
  try {
    var data = await fetchUser();
    print(data);
  } catch (e) {
    print('加载失败：$e');
  } finally {
    print('加载结束');
  }
}
```

### Future 常用方法

```dart
// 等待所有完成
var results = await Future.wait([
  Future.delayed(Duration(seconds: 1), () => 'A'),
  Future.delayed(Duration(seconds: 2), () => 'B'),
]);
print(results);  // [A, B]

// 任一完成即返回
var fastest = await Future.any([
  Future.delayed(Duration(seconds: 1), () => '慢'),
  Future.delayed(Duration(milliseconds: 500), () => '快'),
]);
print(fastest);  // 快

// 等待指定时间
try {
  var result = await fetchUser().timeout(Duration(seconds: 3));
} on TimeoutException {
  print('请求超时');
}

// 延迟执行
await Future.delayed(Duration(milliseconds: 500));
print('500ms 后执行');
```

### whenComplete

类似 `finally`，无论成功失败都执行：

```dart
Future<String> loadWithLoading() async {
  print('加载中...');
  try {
    return await fetchUser();
  } finally {
    print('加载结束');
  }
}
```

## Stream

Stream 是异步事件序列，类似 Future 的"多次版"。

### 创建 Stream

```dart
// 从 Future 创建
var stream = Stream.fromFuture(Future.value('hello'));

// 从可迭代对象创建
var countStream = Stream.fromIterable([1, 2, 3, 4, 5]);

// 周期性生成
var ticker = Stream.periodic(Duration(seconds: 1), (count) => count);

// 使用 StreamController
var controller = StreamController<String>();
controller.add('事件1');
controller.add('事件2');
controller.close();

// 监听
controller.stream.listen((event) {
  print(event);
});
```

### 监听 Stream

```dart
// listen 方式
var subscription = stream.listen(
  (data) => print('数据：$data'),
  onError: (error) => print('错误：$error'),
  onDone: () => print('完成'),
);

// 暂停/恢复/取消
subscription.pause();
subscription.resume();
subscription.cancel();
```

### await for

用同步风格遍历 Stream：

```dart
Future<void> processStream() async {
  await for (var value in countStream) {
    print(value);
    if (value >= 3) break;
  }
}
```

### Stream 转换

```dart
var stream = Stream.fromIterable([1, 2, 3, 4, 5]);

// map — 映射
var doubled = stream.map((n) => n * 2);

// where — 过滤
var evens = stream.where((n) => n.isEven);

// take / skip
var first3 = stream.take(3);     // 取前 3 个
var skip2 = stream.skip(2);      // 跳过前 2 个

// distinct — 去重
var unique = stream.distinct();

// asyncMap — 异步映射
var processed = stream.asyncMap((n) async {
  await Future.delayed(Duration(milliseconds: 100));
  return n * 10;
});
```

### 广播 Stream

默认 Stream 是单订阅（只能 listen 一次），广播 Stream 允许多个监听者：

```dart
var controller = StreamController<String>.broadcast();

controller.stream.listen((e) => print('监听1：$e'));
controller.stream.listen((e) => print('监听2：$e'));

controller.add('hello');
// 监听1：hello
// 监听2：hello
```

## 常见异步模式

### 轮询

```dart
Stream<int> poll(Duration interval) async* {
  var count = 0;
  while (true) {
    await Future.delayed(interval);
    yield count++;
  }
}

poll(Duration(seconds: 1)).listen((count) {
  print('轮询 #$count');
});
```

### 防抖

```dart
StreamTransformer<T, T> debounce<T>(Duration duration) {
  return StreamTransformer.fromHandlers(
    handleData: (data, sink) {
      // 简化版：实际需要 Timer 逻辑
      sink.add(data);
    },
  );
}
```

### 并发请求

```dart
Future<void> loadDashboard() async {
  var results = await Future.wait([
    fetchUser(),
    fetchOrders(),
    fetchNotifications(),
  ]);

  var user = results[0] as User;
  var orders = results[1] as List;
  var notifications = results[2] as List;

  print('仪表盘加载完成');
}
```

## 小结

| 概念 | 说明 |
|------|------|
| `Future` | 一次性的异步结果 |
| `Stream` | 连续的异步事件序列 |
| `async/await` | 异步代码的同步写法 |
| `try/catch` | async 函数中的错误处理 |
| `Future.wait` | 并发等待多个 Future |
| `StreamController` | 手动控制 Stream 的发射 |

## 踩坑提示

::: warning 常见错误
1. **忘记 await** — `var data = fetchUser()` 得到的是 Future 而非数据，必须 `await`
2. **async 函数返回 Future** — 即使 `async int f()` 返回的也是 `Future<int>`
3. **在 build 中直接 async** — Flutter 的 build 方法不能是 async，用 FutureBuilder
4. **Stream 只能听一次** — 单订阅 Stream listen 两次会报错，需要广播 Stream
5. **`Future.wait` 一个失败全部失败** — 需要 `preserveStackTrace` 或单独处理
:::

## 实战：异步读写文件持久化

为 TodoRepository 添加文件持久化：

```dart
import 'dart:io';
import 'dart:convert';

class TodoRepository {
  final List<Todo> _todos = [];
  int _nextId = 1;
  final String _filePath;

  TodoRepository(this._filePath);

  // 从文件加载数据
  Future<void> load() async {
    var file = File(_filePath);
    if (!await file.exists()) return;

    var content = await file.readAsString();
    var json = jsonDecode(content) as List;
    _todos.clear();
    for (var item in json) {
      _todos.add(Todo.fromJson(item));
      if (item['id'] >= _nextId) _nextId = item['id'] + 1;
    }
  }

  // 保存数据到文件
  Future<void> save() async {
    var file = File(_filePath);
    var json = jsonEncode(_todos.map((t) => t.toJson()).toList());
    await file.writeAsString(json);
  }

  // 添加后自动保存
  Future<Todo> add(String content) async {
    var todo = Todo(id: _nextId++, content: content);
    _todos.add(todo);
    await save();
    return todo;
  }

  // 完成后自动保存
  Future<bool> markDone(int id) async {
    var todo = findById(id);
    if (todo != null) {
      todo.markDone();
      await save();
      return true;
    }
    return false;
  }

  // ... 其他方法同上一章
}
```
