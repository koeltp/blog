# 类与面向对象

> 上一章我们用 Map 管理了 Todo 列表，但 Map 缺乏类型约束，容易拼错字段名。本章学习面向对象，把 Todo 封装为真正的类。

Dart 是纯面向对象语言，一切皆对象，所有类都继承自 `Object`。

## 类定义

### 基本结构

```dart
class Person {
  // 实例变量
  String name;
  int age;

  // 构造函数
  Person(this.name, this.age);

  // 方法
  void greet() {
    print('你好，我是 $name，今年 $age 岁');
  }

  // getter
  bool get isAdult => age >= 18;

  // setter
  set updateAge(int newAge) {
    if (newAge > 0) age = newAge;
  }
}

void main() {
  var person = Person('Dart', 10);
  person.greet();         // 你好，我是 Dart，今年 10 岁
  print(person.isAdult);  // false
  person.updateAge = 20;
  print(person.isAdult);  // true
}
```

### 私有成员

Dart 用 `_` 前缀表示库级私有（不是类级私有）：

```dart
class BankAccount {
  String owner;
  double _balance;  // 库内可访问，库外不可

  BankAccount(this.owner, this._balance);

  double get balance => _balance;

  void deposit(double amount) {
    if (amount > 0) _balance += amount;
  }

  void withdraw(double amount) {
    if (amount > 0 && amount <= _balance) {
      _balance -= amount;
    }
  }
}
```

## 构造函数

### 命名构造函数

```dart
class Point {
  double x, y;

  // 默认构造函数
  Point(this.x, this.y);

  // 命名构造函数
  Point.origin() : x = 0, y = 0;
  Point.fromList(List<double> list) : x = list[0], y = list[1];

  @override
  String toString() => 'Point($x, $y)';
}

print(Point(3, 4));            // Point(3.0, 4.0)
print(Point.origin());         // Point(0.0, 0.0)
print(Point.fromList([1, 2])); // Point(1.0, 2.0)
```

### 初始化列表

在构造函数体执行前初始化实例变量，适合 final 字段：

```dart
class Rectangle {
  final double width;
  final double height;
  final double area;

  Rectangle(this.width, this.height) : area = width * height;
}
```

### 工厂构造函数

```dart
class Logger {
  final String name;
  static final Map<String, Logger> _cache = {};

  // 工厂构造函数：可以返回缓存实例
  factory Logger(String name) {
    return _cache.putIfAbsent(name, () => Logger._internal(name));
  }

  // 私有命名构造函数
  Logger._internal(this.name);

  void log(String message) => print('[$name] $message');
}

var logger1 = Logger('App');
var logger2 = Logger('App');
print(identical(logger1, logger2));  // true（同一实例）
```

### 常量构造函数

```dart
class ImmutablePoint {
  final double x;
  final double y;

  const ImmutablePoint(this.x, this.y);

  static const origin = ImmutablePoint(0, 0);
}

// 编译时常量
const p1 = ImmutablePoint(1, 2);
const p2 = ImmutablePoint(1, 2);
print(identical(p1, p2));  // true
```

## 继承

```dart
class Animal {
  String name;

  Animal(this.name);

  void speak() => print('$name 发出声音');
}

class Dog extends Animal {
  String breed;

  Dog(String name, this.breed) : super(name);

  @override
  void speak() => print('$name 汪汪！');

  void fetch() => print('$name 捡球');
}

var dog = Dog('旺财', '柴犬');
dog.speak();   // 旺财 汪汪！
dog.fetch();   // 旺财 捡球
```

### 方法重写规则

- 子类方法用 `@override` 注解
- 重写方法不能缩小可见性
- 重写方法返回类型必须是原类型的子类型

## 抽象类

```dart
abstract class Shape {
  // 抽象方法：没有实现，子类必须重写
  double area();

  // 具体方法：有实现，子类可以继承
  void describe() {
    print('面积：${area().toStringAsFixed(2)}');
  }
}

class Circle extends Shape {
  double radius;
  Circle(this.radius);

  @override
  double area() => 3.14159 * radius * radius;
}

class Rectangle extends Shape {
  double width, height;
  Rectangle(this.width, this.height);

  @override
  double area() => width * height;
}

Shape circle = Circle(5);
circle.describe();  // 面积：78.54
```

## 接口

Dart 没有专门的 `interface` 关键字，每个类都隐式定义了一个接口：

```dart
class Flyable {
  void fly() => print('飞行中...');
}

class Swimmable {
  void swim() => print('游泳中...');
}

// implements：必须重新实现所有方法
class Duck implements Flyable, Swimmable {
  @override
  void fly() => print('鸭子飞');

  @override
  void swim() => print('鸭子游');
}
```

### extends vs implements

| 特性 | extends | implements |
|------|---------|------------|
| 方法实现 | 继承父类实现 | 必须自己实现 |
| 多继承 | 单继承 | 可实现多个接口 |
| 用途 | "是一个"关系 | "能做"关系 |

## Mixin

Mixin 是一种代码复用机制，不涉及继承层次：

```dart
mixin Logging {
  void log(String message) {
    print('[LOG] $message');
  }
}

mixin Timestamp {
  String get timestamp => DateTime.now().toIso8601String();
}

// with 关键字混入
class Service with Logging, Timestamp {
  void doWork() {
    log('开始工作 @ $timestamp');
  }
}

var service = Service();
service.doWork();  // [LOG] 开始工作 @ 2026-06-08T...
```

### Mixin 约束

```dart
// 限制 Mixin 只能用于特定类的子类
mixin Flyable on Animal {
  void fly() => print('$name 起飞！');
}

class Bird extends Animal with Flyable {
  Bird(String name) : super(name);
}

// class Car with Flyable {}  // ❌ Car 不是 Animal 子类
```

## 枚举

```dart
enum Status {
  pending,
  active,
  completed,
}

// 使用
var status = Status.active;
print(status.name);         // "active"
print(status.index);        // 1

// switch 匹配（必须穷举）
switch (status) {
  case Status.pending:
    print('待处理');
  case Status.active:
    print('进行中');
  case Status.completed:
    print('已完成');
}
```

Dart 3 增强枚举：

```dart
enum Planet {
  mercury(0.06, 0.39),
  venus(0.81, 0.72),
  earth(1.0, 1.0),
  mars(0.11, 1.52);

  final double mass;    // 相对地球质量
  final double distance; // 相对地球距离

  const Planet(this.mass, this.distance);

  double get surfaceGravity => mass / (distance * distance);
}

print(Planet.earth.surfaceGravity);  // 1.0
```

## 小结

| 概念 | 关键点 |
|------|--------|
| 构造函数 | 默认/命名/工厂/常量，灵活多样 |
| `@override` | 重写父类方法必须标注 |
| 抽象类 | 定义规范，子类实现细节 |
| 接口 | 每个类都是接口，`implements` 多实现 |
| Mixin | `with` 混入，代码复用利器 |
| 枚举 | Dart 3 支持带属性和方法的增强枚举 |

## 踩坑提示

::: warning 常见错误
1. **构造函数不能继承** — 子类不会自动继承父类构造函数，必须显式调用 `super()`
2. **`_` 是库级私有** — 不是类级私有，同文件内可以访问
3. **Mixin 不是多继承** — `with A, B` 时，后面的 B 会覆盖 A 的同名方法
4. **`const` 构造函数限制** — 所有字段必须是 `final`，不能有函数体
5. **`implements` 要实现全部** — 实现接口时必须重写所有方法和属性
:::

## 实战：封装 TodoRepository 类

用面向对象重构 Todo 应用：

```dart
// Todo 数据类
class Todo {
  final int id;
  final String content;
  bool completed;
  final DateTime createdAt;

  Todo({
    required this.id,
    required this.content,
    this.completed = false,
    DateTime? createdAt,
  }) : createdAt = createdAt ?? DateTime.now();

  void markDone() => completed = true;

  @override
  String toString() => '#$id $content${completed ? " ✓" : ""}';
}

// 仓库类：管理所有 Todo
class TodoRepository {
  final List<Todo> _todos = [];
  int _nextId = 1;

  Todo add(String content) {
    var todo = Todo(id: _nextId++, content: content);
    _todos.add(todo);
    return todo;
  }

  List<Todo> get all => List.unmodifiable(_todos);
  List<Todo> get pending => _todos.where((t) => !t.completed).toList();
  List<Todo> get completed => _todos.where((t) => t.completed).toList();

  Todo? findById(int id) {
    for (var todo in _todos) {
      if (todo.id == id) return todo;
    }
    return null;
  }

  bool markDone(int id) {
    var todo = findById(id);
    if (todo != null) {
      todo.markDone();
      return true;
    }
    return false;
  }
}
```
