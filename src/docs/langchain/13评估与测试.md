## 一、为什么需要评估

LLM 应用输出不确定，传统单元测试不够用。需要专门的评估体系：

| 传统测试 | LLM 评估 |
|---------|---------|
| 输入→精确输出 | 输入→模糊正确的输出 |
| 断言相等 | 断言语义相似 |
| 覆盖率 | 回答质量 |

## 二、LangSmith 追踪

LangSmith 是 LangChain 官方的观测平台，自动追踪每次调用：

### 2.1 配置

```bash
# .env
LANGSMITH_API_KEY=xxx
LANGSMITH_PROJECT=customer-service-bot
LANGCHAIN_TRACING_V2=true
```

配置后，所有 `invoke/stream/batch` 调用自动上报 LangSmith。

### 2.2 追踪内容

每次调用记录：
- 完整的输入/输出
- 每个组件的耗时
- Token 用量和成本
- 中间步骤（Agent 的思考过程）
- 错误信息

### 2.3 自定义追踪

```python
from langchain_core.callbacks import BaseCallbackHandler

class CostTracker(BaseCallbackHandler):
    """追踪每次调用的成本"""

    def on_llm_end(self, response, **kwargs):
        token_usage = response.llm_output.get("token_usage", {})
        total = token_usage.get("total_tokens", 0)
        cost = total * 0.00001  # 假设每 token $0.00001
        print(f"本次调用消耗 {total} tokens，约 ${cost:.6f}")
```

## 三、评估框架

### 3.1 问答评估

```python
from langchain_openai import ChatOpenAI
from langchain.evaluation import QAEvalChain

# 测试数据集
examples = [
    {"question": "退货流程是什么", "answer": "登录APP→我的订单→申请退货→填写原因→提交审核"},
    {"question": "保修期多久", "answer": "电子产品保修1年，配件保修6个月"},
    {"question": "如何联系客服", "answer": "拨打400-xxx-xxxx或APP内在线客服"},
]

# 生成预测
predictions = []
for ex in examples:
    answer = rag_chain.invoke(ex["question"])
    predictions.append({"question": ex["question"], "answer": answer})

# 评估
eval_chain = QAEvalChain.from_llm(ChatOpenAI(model="gpt-4o-mini"))
results = eval_chain.evaluate(examples, predictions)

for i, result in enumerate(results):
    print(f"Q: {examples[i]['question']}")
    print(f"预测: {predictions[i]['answer'][:50]}...")
    print(f"评估: {result['results']}")
    print("---")
```

### 3.2 自定义评估器

```python
from langchain.evaluation import EvaluatorType, load_evaluator

# 语义相似度评估
evaluator = load_evaluator(
    EvaluatorType.QA,
    llm=ChatOpenAI(model="gpt-4o-mini"),
)

result = evaluator.evaluate_strings(
    input="退货流程是什么",
    prediction="登录APP，在我的订单中申请退货即可",
    reference="登录APP→我的订单→申请退货→填写原因→提交审核",
)
print(result)  # {"score": 1, "reasoning": "..."}
```

### 3.3 幻觉检测

```python
from langchain.evaluation import load_evaluator

hallucination_evaluator = load_evaluator(
    EvaluatorType.LABELED_CRITERIA,
    criteria={
        "hallucination": "回答是否包含了参考资料中没有的信息？"
    },
    llm=ChatOpenAI(model="gpt-4o-mini"),
)

result = hallucination_evaluator.evaluate_strings(
    input="退货需要多久",
    prediction="退货审核需要3-5个工作日",  # 如果参考资料没提到3-5天
    reference="退货需提交审核",            # 参考资料原文
)
```

## 四、自动化测试

```python
# tests/test_customer_service.py
import pytest
from app.main import chat

class TestCustomerService:
    """客服系统测试"""

    @pytest.fixture(autouse=True)
    def setup(self):
        self.session_id = "test-session"

    def test_knowledge_query(self):
        """知识问答测试"""
        answer = chat("退货流程是什么", self.session_id)
        assert "退货" in answer
        assert len(answer) > 10

    def test_order_query(self):
        """订单查询测试"""
        answer = chat("查询订单 ORD001", self.session_id)
        assert "ORD001" in answer or "订单" in answer

    def test_out_of_scope(self):
        """超出范围测试"""
        answer = chat("今天天气怎么样", self.session_id)
        # 不应该编造答案
        assert "人工客服" in answer or "无法" in answer or "天气" in answer

    def test_no_hallucination(self):
        """幻觉检测测试"""
        answer = chat("你们卖火箭吗", self.session_id)
        # 不应该编造商品
        assert "火箭" not in answer or "没有" in answer or "不" in answer
```

```bash
# 运行测试
pytest tests/ -v
```

## 五、评估指标

| 指标 | 含义 | 目标 |
|------|------|------|
| 准确率 | 回答正确的比例 | > 90% |
| 相关性 | 回答与问题的相关程度 | > 85% |
| 幻觉率 | 编造信息的比例 | < 5% |
| 响应时间 | 平均响应延迟 | < 3s |
| 工具成功率 | Agent 工具调用成功比例 | > 95% |
